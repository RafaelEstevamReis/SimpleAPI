## Compatibility

Huge compatibility, currently supports:
* .Net 5
* .Net Core 3.1
* .Net Framework 4.5
* .Net Standard 1.1
  * .NetCore 1.0+
  * .Net Framework 4.5
  * Mono 4.6+
  * Xamarin.iOS 10.0+
  * Xamarin.Android 7.0+
  * UWP 8.0+
  * Unity 2018.1+
